Towards a Distributed OS for Data-Intensive Cloud Applications by Stephanie Wang

  • Problem: growing demands for data-intensive applications
  • Solution:
    • HW: horizontal scaling and hw accelerators
    • SW: distributed execution frameworks (data analytics, machine learning)
  • Issues:
    • duplicated system implementation effort
    • reduced framework evolvability
    • difficulty interoperating efficiently between applications, especially when large data is involved

Distributed futures and actors, combined with an end-to-end approach to fault tolerance, can serve as a general-purpose and interoperable execution subtrate for data-intensive applications
